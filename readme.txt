EventPath Middleware.

Sync orders from SER exposition services online storefronts, with SER exposition services offline proprietary ERP (EventPath). 

Background:
This django web application was responsible for pulling orders from an online storefront/ecommerce site, mapping the online orders to the offline copies, then pushing them into SER's proprietary ERP (EventPath).

Data was origanlly processed through a JSON file dumped from the online store and then uploaded into EPMiddleware web application's interface. 

This application was retired with EventPath around 2017.
